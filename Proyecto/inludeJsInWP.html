Como incluir JavaScript en HTML

Para poder incluir o conectar JavaScript a alguno de nuestros proyectos HTML, debemos tener en cuenta que podemos hacerlo de tres formas distinta y dos de ellas comparten el mismo concepto de inclusión.

1. Agregar JavaScript directamente en el archivo HTML

La primera forma consiste en agregar JavaScript de forma directa en el archivo HTML junto con su estructura, para ello debemos escribir la siguiente etiqueta <script></script>, su función es similar a la etiqueta <style></style> Dentro de la etiqueta iremos añadiendo todas las funciones que necesitemos de JavaScript.

¿En qué parte debo agregar le etiqueta?

Se estiman dos opciones que según se estima brinda ciertas ventajas.

La primera de ellas consiste en agregar la etiqueta dentro de la sección de <head> y pueda permanecer separado del contenido.

2. Agregar JavaScript como atributo en las etiquetas de HTML

La segunda forma de como insertar JavaScript en nuestros HTML es por medio de atributos para las diferentes etiquetas que ya posee HTML, pero se tiene que cumplir la condición de que el atributo debe ser en la mayoría de los casos de tipo “Event”, como onclick ya que estos proceden de la estructura JavaScript.

3. Agregar JavaScript como archivo separado

Así como los atributos y selectores insertados con la etiqueta de estilos en el mismo archivo HTML que nos parece un poco desordenado y muy poco practico a la hora de crear una pagina web con mucho contenido que a su vez será para realizar tareas complejas, como un portal de estudiantes, empleados, registro de bancos, etc. Imagínate como sería si a esto le añadimos la sintaxis de JavaScript…

Estamos de acuerdo que no es algo muy sencillo de manejar, por esa razón tanto CSS y JavaScript tienen soporte para tener su sintaxis en un archivo aparte, en este caso JavaScript puede desarrollarse en su propio documento con una extensión .js, de igual forma, debemos especificar su ruta de destino así:

<script src=”archivo.js”></script>

Dejarlo en la carpeta raíz no es lo más idóneo por cuestiones de orden así también debe tener su propia carpeta que comúnmente la mayoría le pone por nombre “js”.
